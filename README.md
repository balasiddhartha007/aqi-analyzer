# 🌍 AQI Analyzer

A real-time air quality monitoring web app built with Python and Streamlit.

## Features
- Live AQI data for any two cities worldwide
- Side by side comparison table
- 7 day PM2.5 forecast trend graph
- Health recommendations based on AQI levels
- City comparison verdict

## Live Demo
👉 https://urban-aqi-analyzer.streamlit.app

## Tech Stack
Python, Streamlit, Requests, Pandas, Matplotlib, WAQI API

## How to Run Locally
pip install -r requirements.txt
streamlit run app.py
